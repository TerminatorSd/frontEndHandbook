# Knowledge of css
> ### 天生inline-block 元素
- \<input>、\<img>、\<button>、\<textarea>、\<label>
> ### rgba()和opacity的透明效果的区别
- opacity 默认继承
- rgba() 默认不继承
- IE9 以下都不支持
> ### px、em、rem 的区别
- px 为固定长度单位
- em 为父元素的font-size 大小
- rem 为根元素的font-size 大小
> ### reset css 和normalize.css
- reset css 是指我们为了消除浏览器默认样式而在项目中引入的全局css
- normalize.css 是reset css 的一种，它提供了一套合理的默认样式值
